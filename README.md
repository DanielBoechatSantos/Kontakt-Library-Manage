O Kontakt Library Manage é um aplicativo para gerencia das livrarias do software Kontakt da Native Instruments.

--------------

Recursos utilizados:
* Visual - PyQT5 (https://pypi.org/project/PyQt5/)
* Manipulação do registro do Windows - WinReg (https://docs.python.org/3/library/winreg.html)
* resources_rc - Para inserir no código, o favicom (ícone) do aplicativo

--------------
Utilização:

Execute o software como administrador.
Selecione a pasta onde está a biblioteca do kontakt
Dê um nome e execute.
